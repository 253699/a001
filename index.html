frps 服务端一键安装

适用于 Linux 服务器,支持 Debian/Ubuntu/CentOS 系统

快速部署 frps 服务端

一行命令即可在您的 Linux 服务器上安装并配置 frps 服务端,支持自动配置防火墙、创建系统服务,安装完成后脚本自动删除。

支持多系统
自动配置
开机自启

一键安装命令

bash <(curl -fsSL https://any001.ipv4.website/sh/a.sh)

* 需要 root 权限,如非 root 用户请在命令前加 sudo

支持的系统

  • Debian / Ubuntu
  • CentOS / RHEL

可选版本

  • frps 0.66.0 (推荐)
  • frps 0.65.0

安装过程说明

  1. 自动检测系统类型并安装依赖
  2. 选择要安装的 frps 版本
  3. 配置服务端参数(端口、token、管理面板等)
  4. 下载并安装 frps 到 /root/AnyFRPS
  5. 创建配置文件和 systemd 服务
  6. 自动配置防火墙规则
  7. 启动服务并设置开机自启
  8. 安装完成后自动删除脚本文件

配置参数

安装过程中会提示您配置以下参数(均有默认值,可直接回车使用默认):

• 服务端口 (默认: 10002)
• Token (默认: 123456)
• Web管理端口 (默认: 10001)
• 管理用户名 (默认: any)
• 管理密码 (默认: 123456)
• 端口范围 (默认: 10006-12000)

常用管理命令

systemctl start frps

启动服务

systemctl stop frps

停止服务

systemctl restart frps

重启服务

systemctl status frps
